# Security

## Reporting vulnerabilities

Report security issues affecting the HelioZero router firmware or its HTTP API through the [HelioZero-ESP32](https://github.com/TheGrimmChester/HelioZero-ESP32) issue tracker when the flaw is in device behaviour.

For issues **only** in this Home Assistant integration (config flow, coordinator, entity handling), open a [private security advisory](https://github.com/TheGrimmChester/HelioZero-HACS/security/advisories/new) or email the maintainer listed on the GitHub profile.

## Please do not

- Paste API bearer tokens, router passwords, or full diagnostics dumps in public issues.
- Use production router credentials in CI logs or screenshots.
