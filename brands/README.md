# Home Assistant brands submission

PNG set for domain `helio_zero`, ready to copy into [home-assistant/brands](https://github.com/home-assistant/brands) as:

`custom_integrations/helio_zero/`

HACS also ships the same files under [`custom_components/helio_zero/`](../custom_components/helio_zero/) (required for HACS default validation and local fallback).

Regenerate from SVG sources:

```bash
npm run prepare:brand
```

Open a PR against `home-assistant/brands` with the contents of `brands/custom_integrations/helio_zero/`. See [brand image specification](https://github.com/home-assistant/brands#image-specification).
